# 맛슐랭(MatChelin) 개인정보 처리방침

**최종 수정일**: 2026-05-01
**시행일**: (서비스 출시일)
**버전**: 1.0 (출시 약관 — Phase 1 무료 운영 기준)

본 문서는 맛슐랭(영문 표기: MatChelin, 이하 "서비스")이 사용자의 개인정보를 어떻게 수집·이용·보관·파기하는지 설명합니다. 한국 「개인정보 보호법」및 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」을 준수합니다.

---

## 1. 개인정보 처리 목적

서비스는 다음 목적을 위해 최소한의 개인정보를 처리합니다:

1. **회원 가입·인증**: Apple Sign In · Kakao 로그인을 통한 본인 확인
2. **서비스 제공**: 맛집 기록 작성·공유, 친구 팔로우, DM, 가볼곳 저장, 위치 기반 추천
3. **알림 발송**: 따봉·댓글·팔로우·DM 등 푸시 알림
4. **고객 지원**: 문의 응대, 분쟁 해결
5. **서비스 개선**: 익명화된 사용 통계 분석

## 2. 수집하는 개인정보 항목

| 분류 | 항목 | 수집 시점 |
|---|---|---|
| 식별자 | Apple `sub` 또는 Kakao `id` | 가입 |
| 프로필 | 닉네임, 자기소개, 출신지, 거주지, 프로필 이미지 (선택) | 가입·편집 |
| 위치 | 정밀 위치 (GPS) — **사용 시점에만, 동의 시** | 현재 위치 버튼·근처 추천 |
| 콘텐츠 | 리뷰 메모, 사진, 별점, 댓글, DM 본문, 가볼곳 메모 | 작성 시 |
| 디바이스 | APNs/FCM 푸시 토큰, OS 버전, 앱 버전, 기기 모델, 로케일 | 권한 허용 시 |
| 사용 통계 | 화면 이동, 버튼 탭, 검색 길이 (Firebase Analytics 익명 집계) | 자동 |

**민감정보 (의료·사상·노조 등)는 일체 수집하지 않습니다.**

**Phase 1 무료 운영**: 본 약관 v1.0 시행 시점에 서비스는 무료로 제공되며, 결제 정보·구독 정보를 수집하지 않습니다. 향후 유료 기능 도입 시 별도 약관 개정 + 사전 공지 + 회원 재동의 절차를 거칩니다.

## 5-2. 이미지 자동 모더레이션 처리 고지

회원이 업로드하는 모든 이미지(리뷰 사진·DM 사진·프로필 사진·의견 첨부)는 **업로드 직전 자동 AI 검사**를 거칩니다. (서비스 이용약관 제5조의 2 정합)

### 처리 항목·목적·보유 기간

| 처리 항목 | 처리 목적 | 처리 위탁사 | 이미지 본체 보유 기간 | 검사 결과 보유 기간 |
|---|---|---|---|---|
| 이미지 binary | NSFW · 폭력 · 마약 · 무기 · 혐오 · CSAM 분류 | Hive AI Inc. (USA) | **저장 X** (검사 즉시 폐기) | 위반 분류만 audit log 영구 보존 |
| 이미지 SHA-256 hash | CSAM hash matching | Microsoft Corporation (USA, PhotoDNA) | **저장 X** | 매칭 시 audit log 영구 보존 |

### 위반 컨텐츠 처리

검사 결과 위반(특히 CSAM 매칭)으로 분류되면:
1. 해당 이미지는 회사 서버에 저장되지 않음
2. 자동 영구 정지 절차 시작 (이용약관 제5조의 2)
3. **한국인터넷진흥원(KISA) 청소년보호 핫라인 24시간 내 신고** (정보통신망법 §44-7, 청소년보호법 §11)
4. 글로벌 영향 시 미국 NCMEC(국립실종아동·착취아동센터) CyberTipline 신고
5. audit 기록(검사 시각·매칭 hash·사용자 ID)은 영구 보존되며 수사기관·법원 영장 시 제출

### 사용자 권리

- 본 자동 모더레이션은 법적 의무 이행을 위한 처리이며 사용자의 명시적 동의를 별도 받지 않습니다 (개인정보보호법 §15(1)6호 — 법령상 의무 준수)
- 위반 분류에 이의가 있는 경우 「의견 보내기」 채널로 문의 가능하나, 실제 CSAM 매칭으로 확인된 경우 정지는 해제되지 않습니다
- audit 기록의 열람을 요구할 권리가 있으나, 운영팀이 사고 evidence 보존 목적으로 제한할 수 있습니다 (개인정보보호법 §35(4))

## 6. 사용자 권리

언제든지 다음 권리를 행사할 수 있습니다:

- **열람**: 프로필 화면에서 확인
- **정정·삭제**: 프로필 편집, 리뷰·댓글·DM 개별 삭제
- **처리 정지·동의 철회**: 앱 내 설정 또는 고객 문의
- **데이터 이동권 (Right to Portability)**: 본인 데이터를 JSON 형식으로 받을 권리 — 설정 → 「의견 보내기」 채널로 요청 시 운영자가 24시간 내 응답 (GDPR Art.20 / CCPA §1798.110 / 정통망법 §30 준수). **본 권리 행사는 무료입니다** (GDPR Art.12(5) / CCPA §1798.130(a)(2))
- **회원 탈퇴**: 프로필 → 설정 → 회원 탈퇴 (기능 도입 시), 문의 (그 전까지)

권리 행사 또는 분쟁 발생 시 **개인정보 보호책임자**에게 문의:
- 책임자: (대표자명)
- 이메일: support@matroad.app
- 앱 내 채널: 설정 → 의견 보내기 (회신 이메일 입력 시 24h 내 답변)
- (기타 연락처 보강)

## 7. 안전조치

- 모든 통신은 HTTPS/TLS 1.2 이상으로 암호화
- 인증 토큰은 iOS Keychain에 저장 (앱 격리 보안 컨테이너)
- 사진은 pre-signed URL을 통해 직접 업로드 — 서버는 메타데이터만 보관
- Apple 표준 암호화 사용 — 사용자 정의 암호화 없음 (Export 면제)
