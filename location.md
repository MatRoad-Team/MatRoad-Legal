# 맛슐랭(MatChelin) 위치정보 이용 약관

**최종 수정일**: 2026-05-01
**시행일**: (서비스 출시일)
**버전**: 1.0 (출시 약관)

본 약관은 「위치정보의 보호 및 이용 등에 관한 법률」(이하 "위치정보보호법")에 따라 맛슐랭(영문 표기: MatChelin, 이하 "서비스")이 이용자의 위치정보를 어떻게 수집·이용·보호하는지 정합니다.

---

## 제1조 (목적)

본 약관은 서비스가 제공하는 **위치기반서비스**의 이용 조건과 위치정보의 수집·이용에 관한 절차 및 이용자 권리를 규정합니다.

## 제2조 (위치정보 수집·이용 목적)

서비스는 다음 목적에 한해 위치정보를 수집·이용합니다:

1. **근처 맛집 추천** — 이용자의 현재 위치 반경 내 친구·맛잘알의 리뷰 제공 (4번째 탭 "오늘 뭐먹지?")
2. **리뷰 작성 시 좌표 기록** — 이용자가 작성한 리뷰의 가게 위치를 지도에 표시 (2번째 탭 "맛지도")
3. **가볼곳 자동 매칭** — 이용자가 저장한 위시리스트와 방문한 가게의 자동 연결

## 제3조 (수집하는 위치정보의 항목)

- **실시간 GPS 좌표** (위도, 경도)
- **정확도** (약 10m ~ 수백 m, iOS가 결정)
- 수집 시점에 한함 — 서버에 실시간 좌표 저장하지 않음 (메모리 처리 후 폐기)

## 제4조 (위치정보 보관 기간)

- **실시간 현재 위치**: 저장하지 않음. 기능 제공 직후 메모리에서 제거
- **리뷰에 첨부된 좌표** (리뷰 작성 시 선택한 가게 위치): 리뷰와 동일한 공개 범위 정책 적용 (서비스 이용 종료·탈퇴 시까지)
- **가볼곳 좌표**: 이용자가 삭제하기 전까지

## 제5조 (이용자의 권리)

이용자는 다음 권리를 행사할 수 있습니다:

1. 위치정보 수집·이용 **동의 거부** (서비스 일부 기능 이용 불가)
2. **동의 철회** (서비스 내 설정 → 위치 권한 OFF 또는 iOS 시스템 설정)
3. 본인의 위치정보 **열람·정정·삭제** 요청
4. 위치정보 **제3자 제공 중단** 요청

권리 행사: support@matroad.app 또는 서비스 내 "내 데이터 내보내기" 기능 이용.

## 제6조 (위치정보 제3자 제공)

서비스는 **위치정보를 제3자에게 제공하지 않습니다**. 단, 다음 예외:

- 법령 근거 요청 (수사·공익 조회)
- 이용자 명시 동의 (향후 기능 제공 시 별도 동의 재요청)

## 제7조 (만 14세 미만 이용자)

서비스는 만 14세 미만 이용자의 가입을 차단합니다. 위치정보 수집 목적상 미성년자 동의 체계를 별도 운영하지 않습니다 (서비스 자체가 만 14세 이상 대상).

## 제8조 (서비스의 책임)

- 위치정보 관리 책임자: (사업자 등록 후 입력)
- 위반 시 연락: support@matroad.app
